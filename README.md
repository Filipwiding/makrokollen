# MakroKollen

En första MVP för att jämföra matvaror utifrån pris och makron, med fokus på studenter, gym, deff och bulk.

## Funktioner i version 0.1

- Sök bland produkter
- Filtrera på kategori
- Sortera på protein per krona
- Sortera på protein per 100 kcal
- Sortera på deff-score och bulk-score
- Produktkort med pris, protein, kcal och källa

## Kör lokalt

Dubbelklicka på `index.html` eller öppna filen i en webbläsare.

## Deploy

Projektet är en statisk HTML-sida och kan läggas upp direkt på Vercel, GitHub Pages eller Netlify.
