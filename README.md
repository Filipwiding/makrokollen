# MakroKollen v6 redesign

Ny layout som inte matchar Matpriskollens röda profil.

## Nytt
- Egen grön/beige färgprofil
- Färgkodade scorekort: grönt/gult/rött
- Deff-score och bulk-score från 1–10
- Score-sektion högt upp på sidan
- Fungerande meny med ankarlänkar
- Produktbilder visas om `image_url` finns i Supabase, annars visas snygga kategori-ikoner
- Butiker filtreras efter vald stad när `stores` och `cities` finns i Supabase

## Ladda upp
Ersätt bara befintlig `index.html` i GitHub med denna fil och gör Commit.
